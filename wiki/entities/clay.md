# Clay

Data enrichment platform. Aggregates 75+ data providers into one platform for [[Waterfall Enrichment]]. [[ColdIQ]] is one of only four Elite Studio Partners globally.

## Key Features

- 75+ data providers in a single UI
- Waterfall enrichment with automatic fallback across providers
- API access for programmatic enrichment at scale
- Claygent AI agent for custom research tasks

## ColdIQ Relationship

[[Michel Lieben]] documented an API integration that reduces Clay's per-lead costs by 70-80% (from $0.15-$0.30 to $0.03-$0.08). One Clay table runs across every [[ColdIQ]] client, handling blocklist, lead sourcing, enrichment, scoring, and routing. See [[Waterfall Enrichment]].

Clay sits at the top of [[ColdIQ]]'s outbound pipeline. Enriched data flows into [[Instantly]], [[Lemlist]], or [[Smartlead]] for sequencing. [[Common Room]] signals and Clay data together drive [[Signal-Based Outbound]].
